JINSE.COM Web前端页面规范文档

写在前面

为了前端开发的团队成员协作高效，以及便于后期维护，现制订一套Web前端规范。规范不是约束，而是服务于设计师与前端开发工程师，减少沟通成本。JINSE.COM PC端网页为固定宽度，不考虑响应式，所以该规范只适用于固定宽度的页面。

CSS

命名规范

  • 头部／header
  • 尾部／footer
  • 部分／section
  • 特定部分／section-xxx
  • 容器／container
  • 竖列／col
  • 宽度为300的列／col-300
  • 1行4等分的单列／col-4
  • 1行6等分的单列／col-6
  • 特定列／col-xxx
  • 盒容器／box
  • 主导航／nav
  • 子导航／nav-sub
  • 特定导航／nav-xxx
  • 菜单／menu
  • 子菜单／menu-sub
  • 行内菜单／menu-inline
  • 特定菜单／menu-xxx
  • 当前激活项／current
  • 栏目块／block
  • 栏目标题／block-title
  • 栏目内容／block-content
  • 栏目内容列表／block-list
  • 无缩紧的栏目块／block-full
  • 无背景色的栏目块／block-no-bg
  • 特定栏目／block-xxx
  • 标签页／tabs
  • 标签页标题栏／tab-title
  • 标签页内容区／tab-content
  • 标签页内容列表／tab-list
  • 特定标签页／tabs-xxx
  • 文章列表／list
  • 文章标题附加栏／mark
  • 文章摘要／summary
  • 日期／date
  • 编辑／editor
  • 标签／tags
  • 面包屑／bread-crumb
  • 分割线／pipe
  • 图标／icon
  • 图标名／icon-editor
  • 图标字体尺寸／icon16
  • 广告条／banner
  • 通栏广告条／banner-full

CSS书写格式

/**
 * 1 - reset
 * 2 - basic
 * 2.1 - common
 * 2.2 - icon
 * 2.3 - pic
 * 2.4 - table
 * 2.5 - menu
 * 2.6 - list
 * 2.7 - tabs
 * 3 - layout
 * 4 - header
 * 5 - footer
 * ----------------------------------------------------------------------------
 */


/**
 * 1 reset
 * ----------------------------------------------------------------------------
 */
 
 html, body, div, h1, h2, h3 ,h4, p, span, em, b, i, ul, ol, li, dl, dt, dd, table, tr, th, td, form, input, lable, select, textarea, img{ margin: 0; padding: 0; border: 0;}
 
 
 /**
  * 2 basic
  * ----------------------------------------------------------------------------
  */
 
body{ min-width: 1200px; line-height: 1.6; font-family: "Helvetica Neue", Helvetica, Arial, "Hiragino Sans GB", "Hiragino Sans GB W3", "WenQuanYi Micro Hei", sans-serif; font-size: 14px; color: #666; background: #f4f6fb;}
h1, h2, h3, h4{ color: #333;}
h1{ font-size: 30px;}
h2{ font-size: 20px;}
h3{ font-size: 16px;}


/* 2.1 common */
.fl{ float: left;}
.fr{ float: right;}
.clear:before, .clear:after, .header:before, .header:after, .footer:before, .footer:after, .container:before, .section:before, .section:after, .container:after, .row:before, .row:after{ display: table; clear: both; content: "";}
.pipe{ margin: 0 10px;}

/* 2.2 icon */
.icon{ margin-right: 8px;}

CSS声明顺序

  1. 位置属性：position, top, right, z-index, display, float...
  2. 大小：width, height, padding, margin...
  3. 文字系列：font, line-height, color, text-align...
  4. 背景：background, border...
  5. 其他：opacity...
